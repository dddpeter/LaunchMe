## 🌟 简介

LaunchMe 是一个现代化的 macOS Launchpad 替代应用，提供了更快速、更灵活的应用启动体验。它具有以下特点：

- **快速搜索** - 实时搜索应用程序，支持名称、Bundle ID、类别等多种搜索方式
- **文件夹管理** - 创建自定义文件夹来组织应用程序
- **拖拽操作** - 直观的拖拽界面，轻松管理应用和文件夹
- **全局快捷键** - 使用快捷键快速呼出应用启动器
- **流畅动画** - 优雅的界面动画和过渡效果

## 🚀 安装和设置

### 系统要求
- macOS 14.0 或更高版本
- 至少 50MB 可用磁盘空间
- 支持的架构：Intel x86_64 或 Apple Silicon

### 安装步骤
1. 下载 LaunchMe 应用程序包
2. 将应用拖拽到应用程序文件夹
3. 首次启动时，可能需要授予以下权限：
   - **辅助功能权限** - 用于全局快捷键功能
   - **文件系统访问** - 用于发现应用程序

### 首次设置
1. 启动 LaunchMe
2. 应用会自动扫描系统中的应用程序
3. 设置全局快捷键（默认：Option + Space）
4. 根据需要调整界面设置

## 🎯 基本使用

### 呼出界面
- **快捷键** - 按下 Option + Space（或自定义快捷键）
- **Dock图标** - 点击 Dock 中的 LaunchMe 图标
- **菜单栏** - 点击菜单栏中的 LaunchMe 图标（如果启用）

### 搜索应用
1. 呼出 LaunchMe 界面
2. 直接输入应用名称进行搜索
3. 搜索结果会实时显示
4. 使用方向键选择应用
5. 按回车键启动应用

### 启动应用
- **单击** - 直接点击应用图标
- **双击** - 双击应用图标（可配置）
- **回车键** - 选择应用后按回车
- **右键菜单** - 右键点击选择"打开"

### 关闭界面
- **Escape键** - 按 Escape 键关闭
- **点击外部** - 点击界面外部区域
- **快捷键** - 再次按下全局快捷键
- **动画关闭** - 等待自动关闭动画

## 🎨 高级功能

### 文件夹管理

#### 创建文件夹
1. **拖拽方式** - 将一个应用拖拽到另一个应用上
2. **右键菜单** - 右键点击应用选择"创建文件夹"
3. **菜单操作** - 使用顶部菜单的"文件 > 新建文件夹"

#### 管理文件夹内容
- **添加应用** - 拖拽应用到文件夹上
- **移除应用** - 从文件夹中拖拽应用到外部
- **重命名文件夹** - 右键点击文件夹选择"重命名"
- **删除文件夹** - 右键点击文件夹选择"删除"

#### 文件夹选项
- **更改颜色** - 为文件夹设置自定义颜色
- **自定义图标** - 为文件夹设置自定义图标
- **排序方式** - 设置文件夹内应用的排序方式

### 拖拽操作

#### 应用到文件夹
1. 按住应用图标不放
2. 拖拽到目标文件夹上
3. 释放鼠标完成移动

#### 文件夹间移动
1. 从源文件夹拖拽应用到外部
2. 拖拽到目标文件夹上
3. 释放鼠标完成移动

#### 重新排序
1. 在同一文件夹内拖拽应用
2. 移动到目标位置
3. 释放鼠标完成排序

### 搜索功能

#### 高级搜索
- **模糊搜索** - 支持拼写错误和部分匹配
- **多字段搜索** - 同时搜索名称、类别、开发者等
- **搜索历史** - 保存最近的搜索记录
- **搜索建议** - 提供智能搜索建议

#### 搜索语法
- **精确匹配** - 使用引号："Safari"
- **排除词** - 使用减号：games -action
- **类别搜索** - 使用冒号：category:productivity
- **开发者搜索** - 使用前缀：dev:apple

### 右键菜单

#### 应用选项
- **打开** - 启动应用程序
- **在访达中显示** - 在 Finder 中显示应用
- **显示包内容** - 显示应用包内容
- **获取信息** - 显示应用详细信息
- **添加到文件夹** - 将应用添加到指定文件夹
- **从文件夹移除** - 从当前文件夹移除应用

#### 文件夹选项
- **打开文件夹** - 展开文件夹内容
- **重命名** - 修改文件夹名称
- **更改颜色** - 设置文件夹颜色
- **设置图标** - 自定义文件夹图标
- **排序方式** - 选择应用排序方式
- **删除文件夹** - 删除空文件夹

## 🔧 故障排除

### 常见问题

#### 快捷键不工作
1. 检查系统偏好设置 > 安全性与隐私 > 隐私 > 辅助功能
2. 确保 LaunchMe 已添加到允许列表
3. 尝试重新设置快捷键
4. 重启应用程序

#### 应用无法启动
1. 检查应用路径是否正确
2. 确认应用权限设置
3. 尝试在访达中直接启动应用
4. 重新扫描应用程序

#### 搜索结果不准确
1. 清除搜索历史
2. 重新扫描应用程序
3. 检查搜索设置
4. 重启应用程序

#### 界面显示异常
1. 检查系统主题设置
2. 调整界面缩放比例
3. 重置界面设置
4. 更新显卡驱动

#### 性能问题
1. 清除图标缓存
2. 减少缓存大小
3. 关闭不必要的动画
4. 重启应用程序

### 日志和诊断

#### 查看日志
1. 打开控制台应用
2. 搜索 "LaunchMe"
3. 查看错误和警告信息
4. 导出日志文件用于分析

#### 性能诊断
1. 启用性能监控
2. 查看性能报告
3. 识别性能瓶颈
4. 优化相关设置

### 重置和恢复

#### 重置设置
1. 关闭应用程序
2. 删除偏好设置文件
3. 重新启动应用程序
4. 重新配置设置

#### 恢复默认配置
1. 在设置中选择"恢复默认"
2. 确认重置操作
3. 重启应用程序
4. 重新个性化设置

## ⌨️ 快捷键参考

### 全局快捷键
- **Option + Space** - 呼出/隐藏 LaunchMe（默认）
- **Cmd + Option + Space** - 呼出并聚焦搜索框
- **Cmd + Escape** - 强制隐藏 LaunchMe

### 界面导航
- **↑↓←→** - 方向键导航
- **Tab** - 在网格和搜索框间切换
- **Enter** - 启动选中的应用
- **Escape** - 关闭界面或返回上级
- **Home/End** - 跳转到开始/结束位置

### 搜索操作
- **Cmd + F** - 聚焦搜索框
- **Cmd + G** - 查找下一个
- **Cmd + Shift + G** - 查找上一个
- **Cmd + A** - 选择所有搜索结果
- **Cmd + R** - 清除搜索

### 文件夹操作
- **Cmd + N** - 新建文件夹
- **Cmd + R** - 重命名选中项
- **Cmd + Delete** - 删除选中项
- **Cmd + D** - 复制选中项
- **Cmd + Option + V** - 移动选中项

### 应用操作
- **Cmd + O** - 打开应用
- **Cmd + I** - 显示应用信息
- **Cmd + Shift + R** - 在访达中显示
- **Cmd + Option + O** - 显示包内容

### 系统操作
- **Cmd + ,** - 打开设置
- **Cmd + Q** - 退出 LaunchMe
- **Cmd + ?** - 显示帮助
- **F1** - 显示快捷键帮助
