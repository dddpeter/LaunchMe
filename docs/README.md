## 📖 项目概述

LaunchMe 是一个现代化的 macOS Launchpad 替代应用，具有以下特点：

### 核心功能
- ✅ **快速搜索** - 实时搜索应用程序
- ✅ **文件夹管理** - 创建自定义文件夹组织应用
- ✅ **拖拽操作** - 直观的拖拽界面
- ✅ **全局快捷键** - 快速呼出应用启动器
- ✅ **流畅动画** - 优雅的界面动画效果

### 技术特点
- 🏗️ **MVVM架构** - 清晰的架构模式
- 🎨 **SwiftUI + AppKit** - 现代化界面技术
- ⚡ **高性能** - 优化的缓存和异步处理
- 🔧 **可扩展** - 模块化设计便于扩展

### 项目状态
- ✅ **开发完成** - 所有核心功能已实现
- ✅ **编译成功** - 项目可以正常编译运行
- ✅ **文档完整** - 完整的技术和用户文档
- ✅ **测试覆盖** - 核心功能单元测试

## 🚀 快速开始

### 环境要求
- macOS 14.0+
- Xcode 15.0+
- Swift 5.9+

### 安装步骤
1. 克隆项目仓库
2. 打开 Xcode 项目文件
3. 选择 LaunchMe scheme
4. 点击 Run 或使用 Cmd+R

### 基本使用
1. 启动应用后，使用 `Option + Space` 呼出界面
2. 直接输入应用名称进行搜索
3. 拖拽应用到文件夹进行组织
4. 右键点击查看更多选项
